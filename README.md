TirarCV

*-Para crear un SuperUser: python3 manage.py createsuperuser. /admin para acceder a su logueo
*-Crear una BD en su gestor MySQL
*-Ejecutar: 
  -python3 manage.py migrate
  -python3 manage.py makemigrations
  -python3 manage.py migrate

*-Para correr el servidor: python3 manage.py runserver
*-Cargar settings.py con los datos de su BD 
*-Crear la carpeta /media
 
 Una vez hecho esto, puede cargar idiomas y tipos de empleos con el administrador.
